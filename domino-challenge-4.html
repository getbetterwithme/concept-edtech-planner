도미노 도전 — 증명 풀이

해밀턴 순환 경로를 이용한 증명

증명할 명제
8×8 체스판에서 서로 다른 색의 칸 2개를 제거하면,
나머지 62칸을 도미노로 항상 채울 수 있다.
Step 1

핵심 아이디어: 해밀턴 순환 경로

8×8 체스판의 64개 칸을 모두 한 번씩만 방문하고 출발점으로 돌아오는 경로를 만들 수 있습니다. 이것을 해밀턴 순환 경로라고 합니다.

8×8 체스판 위의 해밀턴 순환 경로
Step 2

관찰: 경로 위에서 색깔은 번갈아 나타난다

체스판에서 인접한 칸은 항상 서로 다른 색입니다.
따라서 이 순환 경로를 따라가면 밝은색 → 짙은색 → 밝은색 → 짙은색 → ... → 밝은색 → 짙은색 순서로 색깔이 완벽하게 교대합니다.

순환 경로를 고리로 보면: 밝은색-짙은색이 완벽하게 교대합니다 (64칸)
Step 3

다른 색 2칸을 제거하면?

이 긴 줄에서 서로 다른 색 2칸을 하나씩 빼면,
순환 경로가 2개의 경로로 나뉩니다.

핵심은 이것입니다:

  1. 순환 경로에서 밝은색 칸 1개와 짙은색 칸 1개를 제거합니다.
  2. 남은 62칸은 2개의 경로로 나뉩니다.
  3. 밝은색-짙은색이 교대하는 줄에서 다른 색 2칸을 빼면,
    각 경로는 반드시 짝수 개의 칸을 가집니다.
  4. 각 경로에서 연속한 2칸씩 짝지으면 → 그것이 바로 도미노입니다!
Step 4

왜 각 경로가 짝수 개일까?

순환 경로를 일직선으로 펼치면: 밝은색-짙은색-밝은색-짙은색-...-밝은색-짙은색 (64칸)

여기서 밝은색 칸의 위치는 1, 3, 5, 7, ... (홀수 번째)
짙은색 칸의 위치는 2, 4, 6, 8, ... (짝수 번째)

서로 다른 색 2칸을 빼면 하나는 홀수 번째, 다른 하나는 짝수 번째에서 빠집니다.

이 두 점이 원형 줄을 끊어서 만드는 2개의 구간은 각각:

  • 홀수 번째 → 짝수 번째까지: 밝은색-짙은색-...-밝은색-짙은색 → 짝수 개
  • 짝수 번째 → 홀수 번째까지: 짙은색-밝은색-...-짙은색-밝은색 → 짝수 개

따라서 각 경로를 연속 2칸씩 짝지으면 빈틈 없이 도미노로 채울 수 있습니다!

참고

반대로, 같은 색 2칸을 제거하면?

같은 색 2칸을 빼면 — 예를 들어 둘 다 밝은색 칸이면 — 남은 판에는
밝은색 칸 30개, 짙은색 칸 32개가 됩니다.

도미노 1개는 반드시 밝은색 칸 1개 + 짙은색 칸 1개를 덮으므로,
밝은색 칸과 짙은색 칸의 수가 다르면 절대 채울 수 없습니다.

이것은 간단한 세기(counting) 논증으로 증명됩니다.

결론

서로 다른 색 2칸을 제거하면 → 항상 가능
서로 같은 색 2칸을 제거하면 → 항상 불가능

핵심 도구: 해밀턴 순환 경로 + 색깔 교대 성질
